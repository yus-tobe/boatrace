# boatrace

1号艇から6号艇それぞれの選手が過去3年間で1~3着になった確率を入力として与え、そのレースの結果が1-2-3になるかどうかの2値分類を行う。

シミュレーション結果は以下のグラフである。青線が投資した金額、オレンジ線が回収した金額
